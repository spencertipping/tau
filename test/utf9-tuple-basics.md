# Tuple basics
```bash
$ ./pi0 '(1,2,3) (4,5,6) t++ [1 i32+] t* :out
         (1,2,3,4,5,6) [3 % i32%] t* [2 i32>] t% :out'
(5,6,7,2,3,4)
(1,0,1,0)
```
